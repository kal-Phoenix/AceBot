## Uploading Exams
1. Place exam files in the appropriate subject folder under Natural Science or Social Science.
2. Use descriptive filenames, e.g., `Physics_2017_Final.pdf`.
3. Update the backend or database to reference these files.

This structure makes it easy to manage and upload exams by subject and stream.
